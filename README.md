# PaddleRS


PaddleRS是基于飞桨的遥感影像处理平台，支持遥感图像分类，目标检测，图像分割，以及变化检测等常用遥感任务。

### 遥感数据处理
- [ ] 多光谱、高光谱、SAR、无人机RGBD
- [ ] 数据切分、合并、通道提取等功能
- [ ] 支持任意通道数据增强策略
  
### 算法模型
- [ ] 分类（ResNet，MobileNet，HRNet）
- [ ] 检测（PP-YOLO，Faster-RCNN，S2ANet）
- [ ] 分割（DeepLabV3P、OCRNet、UNet）
- [ ] 变化检测（TBD
- [ ] 特有遥感模型（FarSeg）
